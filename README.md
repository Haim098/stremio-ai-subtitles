# Stremio AI Subtitles Add-on 🎬✨

> AI-powered subtitles for Stremio, generated by **Gemini 3 Flash Preview**

## Features

- 🤖 **AI-Generated Subtitles** - Contextual subtitles created by Google Gemini AI
- 🌐 **7 Languages** - Hebrew, English, Arabic, Russian, French, Spanish, German
- ⚡ **Smart Caching** - Generated subtitles are cached to avoid redundant API calls
- 🎯 **Auto-Detection** - Extracts title from filename for better subtitle quality
- 📺 **Movies & Series** - Supports both movies and TV series (with season/episode context)

## Installation

### Install in Stremio

1. Open Stremio
2. Go to **Add-ons** (puzzle icon)
3. Click **Community Add-ons**
4. In the search bar at the top, paste the add-on URL:
   ```
   https://your-deployment-url/manifest.json
   ```
5. Click **Install**

### Run Locally

```bash
# Clone and install
cd stremio-ai-subtitles
npm install

# Start the server
npm start

# Install in Stremio using localhost URL:
# http://localhost:7000/manifest.json
```

## Environment Variables

| Variable              | Description                | Default                                |
|----------------------|----------------------------|----------------------------------------|
| `PORT`               | Server port                | `7000`                                 |
| `GEMINI_API_KEY`     | Google Gemini API key      | (built-in key)                         |
| `RENDER_EXTERNAL_URL`| Auto-set by Render.com     | -                                      |

## How It Works

1. 📺 When you play content in Stremio, the app requests subtitles from all installed subtitle add-ons
2. 🔍 This add-on receives the content ID, type, and filename
3. 🤖 It sends the title to Gemini 3 Flash Preview with a prompt to generate SRT subtitles
4. 📝 Gemini generates contextual dialogue and scene descriptions in the requested language
5. 💾 The SRT file is cached for future requests
6. ✅ Stremio displays the AI-generated subtitles as an option

## 🚨 Important Note

These subtitles are **AI-generated content** based on the title of the media. They are NOT real transcriptions of the audio. They provide contextual, creative subtitles that relate to the content topic.

## Deployment (Render.com - Free)

1. Push to GitHub
2. Connect to [Render.com](https://render.com)
3. Create a new **Web Service**
4. Set Build Command: `npm install`
5. Set Start Command: `npm start`
6. Add environment variable: `GEMINI_API_KEY`
7. Deploy!
